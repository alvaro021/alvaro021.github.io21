Experiencia como estudiante

Durante mi formación universitaria he trabajado en diferentes proyectos, con diferentes Frameworks y herramientas de trabajo, donde me puedo destacar en la captura de requerimientos, arquitectura empresarial y de software mediante Zarchman. Me destaco por un trabajo colaborativo y liderazgo, donde soy capaz de proponer metas y objetivos y desarrollar una solución empresarial arquitectural mediante diagramas de casos de uso, BPMN, system-as-is, system-to-be, diagrama de actividades, de componentes, entre muchos otros.

Algunas de las tecnologías con las que he trabajado son las siguientes:

Laravel y Vue

Ambos frameworks utilizados para el desarrollo de plataformas web utilizando bases de datos relacionales y no relacionales.

MySQL,MariaDB, MongoDB, VoltDB

Manejo de diferentes bases de datos relacionales y no relacionales, de acuerdo con el proyecto que se quiera realizar y las necesidades de éste. Trabajo de Bussines Intelligence mediante el procesamiento de datos con herramientas como Pentaho y diseño de almacenes de datos mediante Data Mart.

Back-end Java

Lenguaje principal trabajado a lo largo de todo mi recorrido como estudiante de la profesión informática.

Front-end HTML 5

He trabajado con diferentes herramientas de desarrollo web como HTML 5, framework Blazor, y los ya mencionados Laravel y Vue.

Arquitecturas

Para proyectos que necesiten los sistemas de tecnologías de la información, he trabajado con arquitecturas de servicios y orientadas a microservicios, además de las ya conocidas arquitecturas monolíticas.

Fortalezas
